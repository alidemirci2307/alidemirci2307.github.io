🔴 Canlı Önizleme

Formu doldurup Politika Oluştur'a tıkla.

2

JSON Kopyala'ya bas ve apps.json dosyasındaki diziye ekle.

3

Değişikliği git add apps.json && git commit -m "yeni uygulama" && git push ile push'la.

4

GitHub Actions otomatik olarak yeni-slug/index.html dosyasını oluşturur ve deploy eder.

5

İstersen HTML İndir ile doğrudan dosyayı alıp slug/index.html olarak kaydet.

⚙ GitHub Ayarları

apps.json dosyasını otomatik güncellemek için GitHub Personal Access Token girin. Token sadece tarayıcınızda saklanır, hiçbir sunucuya gönderilmez.

⚠ Token'ınızı kimseyle paylaşmayın. En az repo yetkisi gereklidir.

Token oluşturmak için: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → "Contents" read+write yetkisi ver.
